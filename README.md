# A1 German → English Anki Deck

876 flashcards for German A1 vocabulary with English translations.

## Download

📥 **[Download from Releases](https://github.com/oprobiu/DataDAF1/releases)** — import directly into [Anki](https://apps.ankiweb.net/)

Also available: [German → Romanian version](https://github.com/oprobiu/DataDAF1/releases/tag/v1.0.0-ro)

## What's in the deck

- 876 cards (1752 with reverse), covering Goethe A1 wordlist
- German word + example sentence on front, English translation on back
- TTS audio for every word and sentence (gTTS, de-DE)
- Color-coded articles: der / die / das
- Tags: NOUN, VERB, ADJ, ADV, PREP, CONJ, PRON, PHRASE, QWORD

## Build from source

Requires Python 3.8+ and [AFAS](https://github.com/oprobiu/AFAS).

```bash
make bootstrap
make validate
make build
```
